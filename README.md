# IntelliPilot
Arduino Due(Cortex-M3) based quadcopter open source project.

### Installation
>>#### to build
`~/IntelliPilot$ make build`
#### to flash
`~/IntelliPilot$ make flash`
#### to screen *via serial port*
`~/IntelliPilot$ make screen`
